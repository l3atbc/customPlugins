jspsych-dropdown plugin

The dropdown plugin displays questions with the options as a dropdown menu. As the user types into the dropdown menu, the plugin narrows down the options based on the user input. The plugin requires the researcher to specify whether the user can select only one option per question, or multiple options.

Parameters

The following table lists the parameters associated with this plugin. Parameters with the default value of undefined must be specified. All of the parameters for each question are enclosed in an object. If there is a single question in a trial, questions is an array that contains one object with all the parameters for that question. If there are multiple questions in the trial, questions becomes an array of objects with each object holding the specific parameters for the question it contains.

NOTE: The following parameters need to be specified for every question separately, even when there are multiple questions in a single trial (see example below). This allows for customizable placeholders for each question. It also makes it possible to have questions that allow multiple selections, and questions that do not allow multiple selections on the same page.

Parameter Type Default Value Description
text string undefined Specifies the text of a question.
options array undefined Specifies the possible answers as separate strings.
allowMultipleSelections boolean undefined Specifies whether multiple selections are allowable for the particular question.
required boolean undefined Specifies whether the question is required or optional.
placeholder string undefined Specifies the placeholder for the dropdown menu.

Data Generated

Name Type Value
rt numeric The response time in milliseconds for the subject to make a response. The time is measured from when the questions first appear on the screen until the subject's response.
preamble string The specific preamble for the whole trial if any (otherwise empty string).
superq string The specific superq for the trial if any (otherwise empty string).
rawResponses JSON string A string in JSON format containing the responses for each question. The encoded object will have a separate variable for the response to each question, with the first question in the trial being recorded in Q0, the second in Q1, and so on. The responses recorded as the name of the option label.

Example

var demographics = {
        type: "dropdown",
        questions: [
{
  text: 'Gender:',
  options: ['Male', 'Female'],
  allowMultipleSelections: false,
  required: true,
  placeholder: 'Select gender'

},
{
  text: 'Your native langauge(s) learned from birth (if not listed choose 'other'):',
  options: ["Chinese", "English", "French", "German", "Italian", "Spanish"],
  allowMultipleSelections: true,
  required: true,
  placeholder: 'Select language(s)'

}]};

Important Note

This plugin relies on the library Choices.js (v 2.8.8). The library needs to be downloaded and included in order for the plugin to work.
